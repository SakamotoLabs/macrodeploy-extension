# MacroDeploy

**AI code audits that run locally — on your own Claude credential.**

MacroDeploy fans out parallel AI subagents across your codebase to surface bugs, security issues, and gaps, then synthesizes them into a clear report — all on your machine, billed to your own Claude plan.

- 🔒 **Runs locally** — your code is analyzed on your machine via Claude Code; nothing runs on our servers.
- 💳 **Your credential, your cost** — use your Claude Pro/Max subscription or an Anthropic API key.
- 🎯 **Expert audit rubrics** — pulled live from MacroDeploy and kept up to date.

Works in **VS Code, Cursor, Windsurf**, and other VS Code-compatible editors.

→ **[macrodeploy.com](https://macrodeploy.com)**

## Quick start

1. Install [Claude Code](https://docs.claude.com/en/docs/claude-code/setup) and sign in (or set an Anthropic API key).
2. Run **MacroDeploy: Code Audit** from the Command Palette.
3. Review the findings in the *MacroDeploy* output channel.

Set up your credential with the **MacroDeploy: Set Up** command — it's stored in your OS keychain, never on disk in plaintext. Model and theme options are available in Settings.
