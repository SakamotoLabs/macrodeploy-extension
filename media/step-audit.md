# Run your first audit

Open a repository, then run **Code Audit (parallel)** — from the editor toolbar
(🛡️), the Command Palette, or the button below.

A panel opens beside your code and runs several audit subagents **in parallel**
(Security, Tests, Code quality, Deploy) — each cheap and fast — then a single
synthesis pass summarizes everything. You'll see each subagent's live progress
and a ticking timer, and can cancel anytime.

Typical run: **1–3 minutes**.
